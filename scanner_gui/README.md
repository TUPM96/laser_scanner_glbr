# 3D Scanner GUI Application

Python GUI application for controlling 3D scanner and visualizing scan data.

## Installation

1. Install Python 3.7 or higher
2. Install required packages:
```bash
pip install -r requirements.txt
```

## Usage

1. Connect your 3D scanner device via USB
2. Run the application:
```bash
python main.py
```

3. Select the serial port from the dropdown
4. Click "Connect" to establish connection
5. Click "Start Scan" to begin scanning
6. View real-time 3D visualization
7. Export scan data to STL file when complete

## Features

- Serial port selection and connection
- Real-time 3D point cloud visualization
- Scan progress tracking
- Export to STL file format
- Stop scan functionality
